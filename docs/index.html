BYKE PROJECT CODE

All Code for byke capstone project
Each microcontroller and interface app has its own repository.

byke interface

Raspberry Pi python app code is in byke interface repo.
byke interface app runs on a raspberry pi zero w and is the main user interface. It controls communication between handle bar mounted buttons and other controllers. It displays bike information on touchscreen and saves data for later viewing.

byke tail light

Taillight pic microcontroller C code is in byke taillight repo
byke tail light controls, brake light, tail light, laser bike lane, and turn signals. Turn signals are three arrows flashing in sequence. Code is written with MPLAB X and MCC. Written for a PIC16F15324

byke motor controller

Motor controller pic microcontroller C code is in byke motor controller repo
byke motor controller controls, motor pwm, connects to current sensor and battery voltage for battery life calculation. Code is written with MPLAB X and MCC. Written for a PIC16F15324

byke battery display

Battery display code is for running a small display which shows battery percentage without starting entire system.
byke battery display is code for an Arduino Pro Mini, with a 1.0" oled display.

byke test app

Test app, is a python app for the raspberry pi.
byke test app is used to test small parts of the final byke interface app. It is also used as a dedugging app. It can read and write all registers on the motor controller and tail light microcontrollers.

byke api test client

API test client, is a python app to test the byke website api.
byke api client, is an app to test the api of the byke webapp. Code from the api test client will be
incorperated into the final byke interface app.
