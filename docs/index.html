      test/debug app

      byke test app is used to test small parts of the final byke interface app. It is also used as a dedugging app. It can read and write all registers on the motor controller and taillight microcontrollers.

      byke api client, is an app to test the api of the byke webapp. Code from the api test client will be incorperated into the final byke interface app.

      webapp

      Webapp for byke is for viewing trip stats. Shows trip route on a map, and graphs trip stats for each trip. An API is how the byke interface uploads trip data to the web app.
